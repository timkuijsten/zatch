NAME

zatch — efficiently watch directory changes on macOS

SYNOPSIS

zatch [-hps] dir ...

DESCRIPTION

zatch efficiently watches directory changes on macOS. It does this by coalescing all changes in a directory that happen within 30 ms and echo a line on stdout to indicate that one or more things changed in that directory or any of it's subdirectories.
The options are as follows:
 
 
-h
Print usage and exit.
 
 
-p
Preflight - echo the name of each dir on the command line once before starting to monitor.
 
 
-s
Show subdirs to narrow down where a change occurred. Note that this ensures a trailing "/" after the echoed dir. Furthermore, since all changes within 30 ms are coalesced, some changes might have occurred in a subdirectory of the one that is echoed.
 
 
-V
Print version and exit.
 
 
dir
One or more directories to watch.

EXIT STATUS

The zatch utility exits 0 on success, and >0 if an error occurs.
